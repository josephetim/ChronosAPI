Chronos API

Your ultimate API for handling all time and date operations.

About the API

The Chronos API provides robust functionality for managing and exploring temporal data, including Unix timestamp conversions, leap year checks, countdowns, duration calculations, and much more.

Features

  • Convert between Unix timestamps and UTC dates.
  • Check if a year is a leap year.
  • Calculate durations between two dates.
  • Retrieve historical events for a specific date.
  • Support for multi-language date formatting.

How to Use

GET /api - Returns the current time in Unix and UTC.
GET /api/:date - Converts a date string or timestamp to Unix and UTC.
GET /api/leap-year/:year - Checks if the year is a leap year.

For more endpoints, refer to the API Documentation
